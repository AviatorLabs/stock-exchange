# Stock Exchange Website

A modular Single Page Application (SPA) that simulates a stock exchange platform where users can browse stocks, manage portfolios, and perform stock transactions. The project is developed collaboratively using HTML, CSS, JavaScript, and Vite.

---

## Features Implemented

### Core Application

- Client-side routing
- Modular project architecture
- Centralized application state
- Component-based UI
- Responsive layouts
- Loading component
- Error component

### Authentication

- Buyer registration
- Seller registration
- Buyer login
- Seller login

### Dashboard

- Buyer dashboard
- Seller dashboard
- Dashboard navigation
- Available stocks component
- Stock holders component
- Buyer portfolio component
- Sell stock form

### User Experience

- Responsive homepage
- Responsive dashboard
- Loading states
- Error handling
